## Tentang Repository

Halo! Perkenalkan, saya Rapsand, mahasiswa Program Studi Informatika. Repository ini dibuat sebagai portofolio pembelajaran selama mengikuti mata kuliah **Data Science**.

Tujuan saya mempelajari Data Science adalah untuk memahami bagaimana data dikumpulkan, dibersihkan, dianalisis, divisualisasikan, hingga digunakan untuk membangun model Machine Learning yang dapat membantu pengambilan keputusan berdasarkan data.

Repository ini berisi seluruh hasil praktikum, latihan, dan tugas dari Pertemuan 1 sampai Pertemuan 7. Setiap pertemuan membahas topik yang berbeda, mulai dari pengenalan Data Science, penggunaan Python untuk analisis data, Data Cleaning, Statistik Deskriptif, Visualisasi Data, Data Preprocessing, hingga Machine Learning menggunakan Regresi Linear.

Melalui repository ini, saya mendokumentasikan proses belajar serta implementasi konsep-konsep Data Science menggunakan Python dan berbagai library pendukung. Repository ini juga menjadi bukti perkembangan kemampuan saya dalam memahami alur kerja Data Science secara bertahap.

# Cara Menjalankan Notebook

## Menggunakan Google Colab

1. Klik link notebook pada tabel di atas.
2. Pilih "Open in Colab".
3. Jalankan setiap sel kode secara berurutan.

## Menjalankan Secara Lokal

### Clone Repository

```bash
git clone https://github.com/USERNAME/data-science-portfolio.git
```

### Masuk ke Folder Repository

```bash
cd data-science-portfolio
```

### Install Library yang Dibutuhkan

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

### Jalankan Jupyter Notebook

```bash
jupyter notebook
```

Kemudian buka file notebook (.ipynb) yang ingin dijalankan.
